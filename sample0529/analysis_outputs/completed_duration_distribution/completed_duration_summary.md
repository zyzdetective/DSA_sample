# 已完成对话的训练时长分析摘要

此分析关注于标记为'已完成' (对话进度 == 'completed') 的对话记录的训练时长分布。

共找到 2244 条标记为'已完成'的对话记录。
在这些记录中，有 **2218** 条具有有效的正训练时长，并用于以下统计分析。

## 主要统计数据：
| 统计指标        | 值 (分钟)     |
|-----------------|---------------|
| 样本数量        | 2218       |
| 平均时长        | 7.56       |
| 标准差          | 7.53       |
| 最短时长        | 0.13       |
| 25%分位点       | 3.57       |
| 中位数 (50%)    | 5.45    |
| 75%分位点       | 8.65       |
| 最长时长        | 102.70       |
