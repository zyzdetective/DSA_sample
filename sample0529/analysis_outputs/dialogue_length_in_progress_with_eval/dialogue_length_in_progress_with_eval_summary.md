# 进行中且有评估的对话 - 对话内容长度分布摘要

此分析关注于'进行中且有评估的对话'类别下的对话记录长度（以Q/A对数量衡量）。

**分析样本数量**: 15760 条有效对话记录（长度 > 0）。

## 主要统计数据：
| 统计指标        | 值 (对话轮次) |
|-----------------|---------------|
| 平均长度        | 9.81       |
| 最小长度        | 1       |
| 25%分位点       | 5       |
| 中位数 (50%)    | 9    |
| 75%分位点       | 13       |
| 最大长度        | 108       |
